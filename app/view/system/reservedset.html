预留主机记录设置

设置预留的主机记录后，普通用户将无法添加这些主机记录的解析。多个主机记录请用逗号分隔，例如：www,@,mail,ftp

说明

  • 预留主机记录是系统级别的设置，对所有域名生效
  • 普通用户（非管理员）无法添加预留的主机记录
  • 管理员不受限制，可以添加任何主机记录
  • 常见需要预留的主机记录：www, @, mail, ftp, admin, api 等
